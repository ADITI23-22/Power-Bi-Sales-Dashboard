# Power BI Sales Dashboard
This is an interactive Power BI dashboard created using a Kaggle sales dataset. The project includes data cleaning, transformation, DAX measures, KPIs and visual insights to analyze sales performance.

# Features & Insights
- Data cleaning and transformation using Power Query
- KPIs: Total Sales, Total Profit, total orders, Total customers, Average Profit Margin .
- Visuals: Profit by Sub-Category, Sales and profit by year, Top 10 Customers, Trend Analysis, sales and Margin by Category,Decomposition tree analysed by sum of                 quantity and explained by segment, state and city.
- Trend analysis by year.

# Dataset Source
Kaggle Retail Superstore Sales dataset (public)

# Tools Used
- Power BI Desktop
- Power Query
- DAX
- Excel
